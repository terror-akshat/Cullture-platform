# Cultural Platform Frontend

React + TailwindCSS + Vite frontend for the Global Culture platform.

## Features

- Browse cultures from different countries and regions
- Filter by region, category, or country
- Add your own cultural experience
- Take quizzes about world cultures
- View leaderboard and top scores

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will open at http://localhost:3000

## Build

```bash
npm run build
```

## Pages

- **Home** - Featured cultures and platform overview
- **Explore** - Browse all cultures with filters
- **Add Culture** - Submit your cultural experience
- **Quiz** - Take quizzes about world cultures
